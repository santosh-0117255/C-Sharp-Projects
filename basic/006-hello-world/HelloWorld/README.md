# Hello World

Basic console output program demonstrating fundamental C# I/O operations and string interpolation.

## Usage

```bash
dotnet run --project HelloWorld.csproj
```

## Example

```
Hello, World!
Welcome to C# programming!
You are learning C# version 12+

What is your name? Alice

Hello, Alice! Nice to meet you!

Press any key to exit...
```

## Concepts Demonstrated

- Console.WriteLine and Console.Write for output
- Console.ReadLine and Console.ReadKey for input
- String interpolation with $""
- Basic null checking with string.IsNullOrEmpty
- Program structure and execution flow
