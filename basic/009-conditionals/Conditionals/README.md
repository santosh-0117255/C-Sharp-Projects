# Conditionals

Demonstrates conditional logic in C#: if/else statements, nested conditions, switch statements, switch expressions, and pattern matching.

## Usage

```bash
dotnet run --project Conditionals.csproj
```

## Example

```
=== If/Else Statements ===
Age: 18
You are an adult.

Temperature: 25°C
It's warm and pleasant.

Score: 85, Member: True
You passed!
Member bonus applied! +10 points
Final score: 95
Grade: A - Excellent!

=== Switch Statement ===
Day 3: Wednesday

Grade B: Great job!

=== Switch Expression (C# 8+) ===
Day 3 (switch expression): Wednesday
Grade B (switch expression): Great job!

=== Pattern Matching with if ===

Data is a string: "Hello, World!" (Length: 13)

Alice is an adult living in London.

User input is null.

=== Conditional (Ternary) Operator ===
Max of 10 and 20: 20
Sum status: High
```

## Concepts Demonstrated

- Simple if statements
- if/else if/else chains
- Nested if statements
- Classic switch statement with break
- Multiple cases in switch (fall-through)
- Switch expressions (C# 8+)
- Pattern matching with is keyword
- Property pattern matching
- Null checking with patterns
- Ternary conditional operator
